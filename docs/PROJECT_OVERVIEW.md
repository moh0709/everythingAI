# PROJECT OVERVIEW

EverythingApp is designed as a **local-first AI-powered file brain** that transforms raw files into structured, searchable, and actionable knowledge.

## Vision

To create a system that combines:

- The speed of file indexing (like Everything)
- The intelligence of AI (LLMs)
- The structure of a knowledge base (Wikipedia-style)
- The power of automation (file organization)

## Core capabilities

1. Full-drive indexing
2. Content understanding (documents, PDFs, etc.)
3. Semantic search
4. Chat with your files
5. Knowledge base generation
6. AI-assisted file organization
7. Safe execution of file operations

## Product layers

1. File Layer → Raw filesystem
2. Index Layer → Metadata + structure
3. Knowledge Layer → Extracted meaning
4. AI Layer → Reasoning + decisions
5. Action Layer → File operations

## Target users

- Developers
- Power users
- Enterprises
- Knowledge workers

## Key principle

EverythingApp must be:

- Local-first
- Safe
- Transparent
- Reversible (undo actions)
